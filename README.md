# rxjs-dash

Generate [RxJS](https://github.com/Reactive-Extensions/RxJS) docset for [dash](https://kapeli.com/dash)

## Generate docset

Make sure you have `node`, `bower` on your system.

```
$ git clone riiid/rxjs-dash --recursive
$ cd rxjs-dash
$ npm start
```

see `rxjs.docset`
